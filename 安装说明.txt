lyj · AI求职智能体 v1.0.1

安装方法：
1. 解压这个 ZIP 文件。
2. 打开 Edge：edge://extensions
   Chrome 用户打开：chrome://extensions
3. 开启“开发者模式”。
4. 点击“加载解压缩的扩展”。
5. 选择解压后的”lyj · AI”文件夹。
6. 打开扩展侧边栏，填写你自己的 DeepSeek API Key、简历和筛选条件。

注意：
- 请不要使用别人的 DeepSeek API Key。
- 本工具会自动操作 BOSS 直聘页面，正式投递前请认真审核岗位。
- 改进版新增了最低薪资和岗位过滤黑名单，支持“不含销售性质”等否定语境识别。
